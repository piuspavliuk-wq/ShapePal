# ShapePal

Fitness app: **workouts**, **plan**, **calorie counter**, and **transformation** tracking. Built with Next.js and Tailwind CSS.

## SEO

- Meta title/description target: **ShapePal**, workouts, plan, calorie counter, transformation, fitness
- JSON-LD (WebApplication) for search
- Open Graph and Twitter cards
- `sitemap.xml` and `robots.txt` via App Router
- US English (`lang="en-US"`)

Set `NEXT_PUBLIC_SITE_URL` to your production URL (e.g. `https://shapepal.com`) for canonical URLs and sitemap.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
